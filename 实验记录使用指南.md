# 实验记录系统使用指南

## 概述

本实验记录系统旨在帮助您系统地记录算法运行的结果、数据和实验过程。系统采用混合方案：
- 每个实验生成独立的JSON记录文件
- 所有实验的摘要信息记录到中央索引文件
- 支持记录算法ID、参数、最终结果和迭代过程

## 文件结构

```
experiment_logger.py      # 实验记录器主模块
experiment_logs/          # 实验记录目录（自动创建）
  ├── experiment_index.json    # 中央索引文件
  └── *.json              # 各个实验的详细记录
实验记录使用指南.md       # 本指南
```

## 快速开始

### 1. 导入实验记录器

在您的Python脚本中添加以下代码：

```python
import sys
import os
sys.path.append(os.path.dirname(os.path.abspath(__file__)))
from experiment_logger import create_experiment_logger
```

### 2. 创建实验记录器

```python
# 创建实验记录器
logger = create_experiment_logger(
    experiment_name="您的实验名称",  # 可选，默认自动生成
    log_dir="experiment_logs"       # 可选，默认"experiment_logs"
)

# 设置算法ID
logger.set_algorithm_id("PSO")  # 例如：PSO, BA, GA-PSO-BA等

# 记录算法参数
logger.set_parameters({
    "w": 0.8,
    "c1": 1.5,
    "c2": 1.5,
    "population_size": 30,
    "max_iterations": 100,
    "dataset": "Hokkaido_Consumption"
})
```

### 3. 在迭代过程中记录数据

#### 方法一：在算法循环中添加记录

```python
def pso_algorithm(..., logger=None):
    # ... 初始化代码 ...
    
    for i in range(max_iterations):
        # 开始迭代计时
        if logger:
            logger.start_iteration()
        
        # ... 算法迭代代码 ...
        
        # 计算当前最优值
        current_best = calculate_best()
        
        # 记录迭代信息
        if logger:
            logger.record_iteration(
                iteration=i,
                cost_value=float(current_best),
                best_solution=best_solution.tolist(),  # 转换为列表
                additional_info={"avg_fitness": float(avg_fitness)}
            )
    
    return final_result, best_solution
```

#### 方法二：记录迭代历史数组

如果您的算法已经记录了历代最优值（如`gb`数组），可以在算法结束后批量添加：

```python
# 假设gb是历代最优值的数组
for i, cost in enumerate(gb):
    logger.record_iteration(
        iteration=i,
        cost_value=float(cost)
    )
```

### 4. 记录最终结果

```python
# 运行算法
final_cost, best_solution, iteration_history = run_algorithm(logger=logger)

# 记录最终结果
logger.record_final_results(
    final_cost=float(final_cost),
    best_solution=best_solution.tolist(),
    forecast_values=forecast_values,  # 预测值
    mape=mape_value,                  # MAPE误差
    optimal_parameters={              # 最优参数
        "alpha": alpha,
        "beta": beta,
        "gamma": gamma
    }
)
```

### 5. 保存记录

```python
# 保存实验记录到JSON文件
log_file = logger.save()  # 自动生成文件名

# 更新中央索引
logger.update_central_index()

print(f"实验记录已保存到: {log_file}")
```

## 集成到现有脚本的示例

### 简单集成示例

以下是在现有算法脚本中添加实验记录的最小修改：

```python
# 在文件开头添加导入
import sys
import os
sys.path.append(os.path.dirname(os.path.abspath(__file__)))
from experiment_logger import create_experiment_logger

# 在脚本主函数中创建记录器
if __name__ == "__main__":
    # 创建实验记录器
    logger = create_experiment_logger("PSO_HoltWinters")
    logger.set_algorithm_id("PSO")
    
    # 记录参数
    logger.set_parameters({
        "N": N,
        "D": D,
        "slen": slen,
        "n_preds": n_preds
    })
    
    # 修改算法调用，传递logger参数
    # 需要修改psoAlgorithm函数定义，添加logger参数（可选）
    result = psoAlgorithm(w, c1, c2, x, N, D, slen, n_preds, logger=logger)
    
    # 记录最终结果
    logger.record_final_results(
        final_cost=float(result[0]),
        best_solution=result[1].tolist()
    )
    
    # 保存记录
    logger.save()
    logger.update_central_index()
```

## 数据分析

### 1. 读取实验记录

```python
import json

with open("experiment_logs/experiment_1234_20240115_103000.json", "r") as f:
    data = json.load(f)

# 提取迭代历史
iterations = [item["iteration"] for item in data["iteration_history"]]
costs = [item["cost_value"] for item in data["iteration_history"]]

# 绘制收敛曲线
import matplotlib.pyplot as plt
plt.plot(iterations, costs)
plt.xlabel("迭代次数")
plt.ylabel("代价函数值")
plt.title("算法收敛曲线")
plt.show()
```

### 2. 分析多个实验

```python
import pandas as pd
import json
import glob

# 读取所有实验文件
experiment_files = glob.glob("experiment_logs/*.json")
experiments = []

for file in experiment_files:
    with open(file, "r") as f:
        data = json.load(f)
        experiments.append({
            "experiment_id": data["experiment_id"],
            "algorithm": data["algorithm_id"],
            "final_cost": data["results"]["final_cost"],
            "duration": data["results"]["total_duration"],
            "parameters": str(data["parameters"])
        })

# 创建DataFrame进行比较
df = pd.DataFrame(experiments)
print(df.sort_values("final_cost"))
```

## 常见问题

### 1. 如何避免记录过多数据？

- 只记录关键迭代（如每10次迭代）
- 记录摘要统计而非完整数据
- 使用`record_iteration`的可选参数控制记录内容

### 2. 如何比较不同算法的结果？

使用中央索引文件快速比较：

```python
import json

with open("experiment_logs/experiment_index.json", "r") as f:
    index = json.load(f)

# 按算法类型分组
from collections import defaultdict
algorithms = defaultdict(list)
for exp in index:
    algorithms[exp["algorithm_id"]].append(exp["final_cost"])

# 计算每个算法的平均性能
for algo, costs in algorithms.items():
    print(f"{algo}: 平均代价 = {sum(costs)/len(costs):.4f}")
```

### 3. 如何将现有CSV输出与JSON记录结合？

```python
# 保持原有的CSV输出
```
